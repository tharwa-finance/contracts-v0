### Tharwa Token (TRWA)


- TRWA is the Tharwa governance and incentive token
- TRWA is a LayerZero Omnichain Fungible Token (OFT)
- TRWA is a Pausable token for security purposes

to install dependencies run :

```bash
pnpm install
```

to deploy TRWA run :

```bash
npx hardhat lz:deploy
```
to run tests run :

```bash
forge test
```
